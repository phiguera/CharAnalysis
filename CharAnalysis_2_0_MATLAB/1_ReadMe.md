# CharAnalysis 2.0

This directory contains the Version 2.0 MATLAB implementation of *CharAnalysis*. 
For full program description, getting started instructions, and citations, see the 
[main README](https://github.com/phiguera/CharAnalysis/blob/master/README.md).

## Directory Structure
- `/Functions` — modernized MATLAB `.m` files with improved input validation
- `/Documentation` — updated User's Guide and the *CharAnalysis Modernization Report*
- `/Testing` — scripts for comparing V2.0 outputs against V1.1 reference results

## Validation
The script `z_Compare_CharAnalysis_V1_V2.m` is provided for comparing Version 2.0 
outputs against Version 1.1 reference results. Users are advised to run this 
comparison on their own datasets to verify that results are consistent between versions.
