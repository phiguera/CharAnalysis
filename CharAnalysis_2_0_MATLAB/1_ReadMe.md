# CharAnalysis 2.0

This directory contains the Version 2.0 MATLAB implementation of *CharAnalysis*. 
For full program description, getting started instructions, and citations, see the 
[main README](https://github.com/phiguera/CharAnalysis/blob/master/README.md).

## Directory Contents
- MATLAB `.m` files — the modernized CharAnalysis 2.0 codebase
- `z_Compare_CharAnalysis_V1_V2.m` — script for comparing V2.0 outputs against V1.1 reference results
- Example datasets — `.csv` input files for the Code Lake (CO) and Chickaree lake (CH10) sites
- V1.1 reference results — `..._V_1_1...` `.csv` files used for version comparison
- Validation report — `.md` file documenting V2.0 validation against V1.1 outputs

## Validation
Users are advised to run `z_Compare_CharAnalysis_V1_V2.m` on their own datasets 
to verify that results are consistent between versions. The Code Lake and Chickaree Lake 
datasets and their V1.1 reference files are provided as examples.
