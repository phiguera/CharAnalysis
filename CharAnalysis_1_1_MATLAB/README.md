# CharAnalysis 1.1 — MATLAB Implementation

This directory contains the original Version 1.1 MATLAB source code for
*CharAnalysis*, retained for reference. **Users are encouraged to use Version 2.0**,
which is located in [`CharAnalysis_2_0_MATLAB/`](https://github.com/phiguera/CharAnalysis/tree/master/CharAnalysis_2_0_MATLAB).
The analytical methods are unchanged between versions.

## Requirements

- MATLAB 7.0 or higher
- MATLAB Curve Fitting Toolbox (required for `smooth()` function calls)

## Directory Contents

**Source code**
- `CharAnalysis.m` — main entry point; run this to start the program
- All other `.m` files — supporting functions called by `CharAnalysis.m`

## Documentation

The original User's Guide for Version 1.1 is available as
[ARCHIVED_UsersGuide_v_0.9_2009.pdf](https://github.com/phiguera/CharAnalysis/blob/master/ARCHIVED_UsersGuide_v_0.9_2009.pdf)
